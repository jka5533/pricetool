One-time setup — get running in 3 steps
1
Install Ollama
Download from ollama.com and run the installer (Windows / Mac / Linux)
2
Download the vision model (one time, ~4 GB)
ollama pull llava
Wait for download to complete. For low-RAM PCs use: ollama pull llava:7b
3
Start Ollama (if not already running)
ollama serve
Then click the status pill above to refresh — it should turn green.
